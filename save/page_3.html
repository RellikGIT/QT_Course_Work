Глава 3 - Алгоритм Штрассена

     Алгоритм, приведённый выше имеет ассимптоматическую сложность O(n*m*q) – то есть, при увеличении всех размеров матриц в 2 раза, скорость умножения уменьшится в 8 раз. Для ускорения умножения были разработаны различные алгоритмы, усовершенствующие стандартный. Одним из примеров такого алгоритма является алгоритм Штрассена, разработанный Фолькером Штрассеном в 1969 году. Данный алгоритм применим к квадратным матрицам, размер которых является степенью двойки.

     Пусть A и B – две квадратные матрицы размером n x n. Матрица C – произведение AB. Если n не является степенью двойки, матрицы дополняются нулями до ближайшего подходящего размера. Матрицы A, B, C разделяются на 4 равные матрицы следующим образом:






















     Остальные матрицы разделяются аналогичо. При этом
C1 = A1 B1 + A2 B3
C2 = A1 B2 + A2 B4
C3 = A3 B1 + A4 B2
C4 = A3 B2 + A4 B4

     Теперь определим новые элементы
P1 = (A1 + A4)(B1 + B4)
P2 = (A3 + A4)B1
P3 = A1(B2 – B4)
P4 = A4(B3 – B1)
P5 = (A1 + A2)B4
P6 = (A3 - A1)(B1 + B2)
P7 = (A2 - A4)(B3 + B4)

     Элементы матрицы C вычисляются по формулам ниже
C1 = P1 + P4 – P5 + P7
C2 = P3 + P5
C3 = P2 + P4
C4 = P1 – P2 + P3 + P6

     Рекурсивный процесс продолжается n раз, до тех пор пока размер матриц Ci не станет равным единице, где умножение матриц есть умножение самих элементов.
